# Fintech Analytics Dashboard

## 📊 Overview
This project demonstrates an end-to-end fintech analytics solution using Snowflake and Power BI. It analyzes revenue performance, fraud risk, and customer behavior through interactive dashboards.

---

## 🛠 Tech Stack
- Snowflake (Data Warehouse)
- SQL (Data Cleaning & Transformation)
- Power BI (Dashboard)
- DAX (KPIs)

---

## 🔄 Data Pipeline
RAW → STG → FCT → Power BI

---

## 📈 Dashboard Pages

### Executive Overview
- Revenue trends
- Growth rate
- Transaction insights

### Risk & Fraud
- Dispute rate
- Fraud score trends
- VPN vs fraud analysis

### Customer Insights
- Customer lifetime value
- Repeat customers
- Top customers

---

## 🔍 Key Insights
- Higher fraud risk observed in VPN-based transactions
- Revenue concentrated among top customers
- Dispute trends highlight risk patterns
